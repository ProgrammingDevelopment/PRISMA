# Folder Template Surat Menyurat RT 04

Folder ini berisi template-template surat resmi RT 04 yang dapat diunduh oleh warga.

## Daftar Template Tersedia

1. **Surat Keterangan Kematian** - `kematian.docx` / `kematian.pdf`
2. **Surat Keterangan Tidak Mampu (SKTM)** - `sktm.docx` / `sktm.pdf`
3. **Surat Pengantar Pindah Domisili** - `pindah.docx` / `pindah.pdf`
4. **Surat Keterangan RT (Umum/Kelakuan Baik)** - `umum.docx` / `umum.pdf`
5. **Surat Keterangan Domisili** - `domisili.docx` / `domisili.pdf`
6. **Laporan Keamanan** - `keamanan.docx` / `keamanan.pdf`

## Cara Penggunaan

1. Download template yang dibutuhkan
2. Isi data yang diperlukan pada template
3. Cetak dan serahkan ke pengurus RT untuk ditandatangani

## Catatan

- Semua template dalam format DOCX dapat diedit menggunakan Microsoft Word atau Google Docs
- Format PDF tersedia untuk referensi atau jika perlu dicetak langsung
- Silakan hubungi sekretariat RT jika ada pertanyaan
